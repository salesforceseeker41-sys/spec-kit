# LWC Standards

## Purpose

Define Lightning Web Component standards for accessibility, performance, security, and maintainability.

## Ownership

Owned by the Platform Team.

## Starter Guidance

- Use Lightning Base Components when practical.
- Protect server calls with Apex security enforcement.
- Handle loading, empty, error, and permission states.
- Follow accessibility expectations for keyboard and screen reader use.
- Keep components focused and reusable.
